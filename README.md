## 📖 Sobre

Este repositório contém **exercícios e pequenos projetos desenvolvidos durante o curso de Java do SENAI**.
O objetivo principal é praticar os fundamentos de Java e os conceitos de Programação Orientada a Objetos (POO).

---

## 🧰 Tópicos Abordados

* Sintaxe e convenções do Java
* Variáveis e tipos de dados
* Estruturas condicionais (`if`, `switch`)
* Laços de repetição (`for`, `while`, `do-while`)
* Métodos e funções
* Programação Orientada a Objetos (Classes, Objetos, Herança, Polimorfismo)
* Exercícios e pequenos projetos práticos

---

## 🚀 Projetos / Exercícios

* **Exercícios Básicos:** Hello World, operações matemáticas, loops, condições
* **POO:** Classes, objetos e métodos simples
* **Projetos de Prática:** Mini aplicativos para reforçar o aprendizado

---

## 📌 Objetivo

Este repositório tem como objetivo **aprender, praticar e compartilhar código** durante o curso de Java.
